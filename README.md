# onlygo-app [![Build Status](https://travis-ci.org/heptastique/onlygo-app.svg?branch=master)](https://travis-ci.org/heptastique/onlygo-app)

## Setup

### Pré-requis
1. NodeJS avec npm
2. Ionic installé avec la commande
```
npm install -g ionic
```

### Installation
```
npm install
```

## Exécution
```
ionic serve --lab
```
